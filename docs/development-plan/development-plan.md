# Development Plan

This file is intended to track planned, active, completed, and blocked work.

Use simple statuses so agents and contributors can coordinate without guessing.

Recommended statuses:

- `pending`
- `in_progress`
- `done`
- `blocked`

## Suggested Columns Or Fields

- ID.
- Title.
- Description.
- Owner or role.
- Status.
- Dependencies.
- Acceptance criteria.
- Verification.
- Notes.
