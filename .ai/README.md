# AI Context Layer

This folder contains project-local AI context guidance.

Use it to help agents find the right source files without loading the whole repository.

## Files

- `context-routing.md`: how to choose the smallest useful context path.
- `prompt-recipes.md`: reusable prompts for common project work.
- `maps/`: compact maps by task area.
- `index/`: generated index guidance. Generated JSON files should not be committed unless the project explicitly chooses to track them.
