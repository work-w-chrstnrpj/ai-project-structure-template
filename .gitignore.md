# Git Ignore Placeholder

Create a real `.gitignore` when the project chooses its language, runtime, tooling, and build process.

At minimum, ignore:

- Dependency folders.
- Build outputs.
- Test and coverage outputs.
- Runtime logs.
- Local environment files.
- Local database files.
- Generated analysis outputs.
- Temporary editor and operating-system files.

Do not ignore source files, documentation, templates, or intentional examples.
