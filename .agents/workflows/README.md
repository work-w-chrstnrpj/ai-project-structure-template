# Workflows

| Workflow | Description |
| --- | --- |
| `feature-delivery` | Product intent through planning, implementation, tests, docs, and release check. |
| `bugfix-delivery` | Investigation through fix, regression tests, docs when needed, and release check. |
| `refactor-delivery` | Structure change with behavior-preserving tests, review, and release check. |
| `chore-delivery` | Risky chores through planning, execution, verification, and config/doc notes. |
| `release-readiness` | Release check, verification evidence, and blocker summary. |
| `investigation-to-rca` | Investigation through evidence review to formal RCA when needed. |
