## Root Commands

Replace these placeholders with real commands:

- `[setup command]`: install dependencies or prepare the local environment.
- `[dev command]`: start the main local development process.
- `[lint command]`: run style, formatting, or static quality checks.
- `[typecheck command]`: run type, compile, or static analysis checks when applicable.
- `[test command]`: run the main automated test suite.
- `[build command]`: create the production or release artifact.
- `[verify command]`: run the release-ready verification bundle.
